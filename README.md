# Projet de POO 2022

## Usage
### Version graphique
Si vous avez ant, il suffit de faire la commande:
```bash
ant
```

Sinon, allez dans le répertoire src et faites la commande:
```bash
javac App.java && java App
```

### Version terminale
Si vous avez ant, il suffit de faire la commande:
```bash
ant term
```

Sinon, allez dans le répertoire src et faites la commande:
```bash
javac App.java && java App term
```
